tag descrição
< hr > Define uma quebra temática em uma página HTML (como um shift ou um tópico).
< frameset > Usada para guardar elementos < frame >.
< thead > Agrupa conteúdo de cabeçalho em uma tabela HTML.
< tt > Cria textos que são formatados como se fossem apresentados em displays como teletipo e terminal.
< footer > Define um rodapé para um documento ou seção.
< area > Define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis).
< b > Especifica um texto em negrito sem qualquer importância adicional.
< div > Define uma divisão ou uma seção em um documento HTML.
< sub > Define texto subscrito.
< style > Define informação de estilo (CSS) para um documento.
< dl > Define uma lista de descrição.
< blockquote > Especifica uma seção que é uma citação de alguma outra fonte.
< dialog > Define uma caixa de diálogo ou subjanela.
< legend > Define um título para o elemento < fieldset >.
< cite > Define o título de um trabalho criativo.
