Para que serve?

A tag <thead> é usada para agrupar conteúdo de cabeçalho em uma tabela HTML. O elemento <thead> é usado em conjunto com os elementos <tbody> e <tfoot> para especificar cada parte da tabela (cabeçalho, corpo, rodapé). Navegadores podem usar esses elementos para habilitar a rolagem pelo corpo da tabela independentemente do cabeçalho e rodapé. Além disso, quando imprimindo uma larga tabela que possui muitas páginas, esses elementos podem permitir imprimir o cabeçalho e rodapé no topo e base de cada página.
