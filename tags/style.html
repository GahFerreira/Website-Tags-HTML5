Para que serve?

A tag <style> é usada para definir informação de estilo (CSS) para um documento. Dentro do elemento <style> você especifica como os elementos HTML devem ser formatados no navegador.

Atributos da tag
atributo valor descrição
media media_query Especifica a mídia/dispositivo para qual a fonte de mídia é otimizada.
type text/css Especifica o tipo da mídia na tag <style>
