Para que serve?

A tag <area> define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis). Elementos <area> são sempre aninhados dentro de uma tag <map>.

Atributos da tag
. . . .
atributo valor descrição
alt text Especifica um texto alternativo para a área. Requerido se o atributo href está presente.
coords coordinates Especifica as coordenadas da área.
download filename Especifica que o alvo vai ser baixado quando o usuário clicar no hyperlink.
href URL Especifica o hyperlink alvo para a área.
hreflang language_code Especifica a linguagem da URL alvo.
media media query Especifica a mídia/dispositivo para qual a URL é otimizada.
