Para que serve?

A tag <dialog> define uma caixa de diálogo ou subjanela. O elemento <dialog> torna mais fácil criar popups e modais em uma página web.

Atributos da tag
atributo valor descrição
open open Especifica que o elemento dialog está ativo e que o usuário pode interagir com ele

Exemplo de uso da tag
<dialog open>This is an open dialog window</dialog>
