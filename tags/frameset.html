Para que serve?

A tag <frameset> é usada para guardar elementos <frame>

ESSA TAG NÃO É MAIS COMPATÍVEL NO HTML 5.

Atributos da tag
atributo valor descrição
cols number Especifica o número e o tamanho de espaços horizontais em um frameset
rows number Especifica o número e o tamanho de espaços verticais em um frameset
